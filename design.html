06 Motion / Animação

Todas as animações usam GSAP 3.12.5 com ScrollTrigger. O scroll suave é gerenciado pelo Lenis 1.1.13 (lerp: 0.08), integrado ao ticker do GSAP.

Animação Ease Duração Uso
Preloader bar power2.inOut 1.8s Progress bar + contador 0→100
Preloader exit expo.inOut 0.9s yPercent: -100, slide out
Hero entrance expo.out 1.1s · stagger 0.1 Clip reveals, y: 100→0
Scroll line pulse power1.inOut 1.4s · yoyo ∞ scaleY: 1→0.25 loop
sr-reveal power3.out 0.85s Seções claras, y: 28→0, op: 0→1
sr-dark power3.out 0.85s About section, mesmo parâmetro
proj-row stagger power3.out 0.7s · delay i×0.07 y: 35→0, op: 0→1
Counters power2.out 1.6s 0 → target, snap integer
Hero parallax none (scrub) scrub yPercent: 0→18
Cursor dot — 0.08s Segue mouse diretamente
Cursor ring lerp 0.1 ticker Segue com lag suave
Lang toggle power2.out 0.3s opacity 0→1, y -5→0
Underline hover CSS cubic-bezier .25,1,.5,1 0.35s width 0→100% no ::after
